SwiftRides

SwiftRides is a car rental Web Application i developed using Django as the backend framework and HTML, Tailwind CSS, and JavaScript as frontend technologies. It is cross-platform (not dependent on OS), can be accessed via a web browser. SwiftRides has two user sides - rental companies and customers (renters). SwiftRides allows for seamless profile management, car upload and management, car reservation and reservation management.
